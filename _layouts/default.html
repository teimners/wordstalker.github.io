О копипастах
Генератор ASCII изображений по картинке
Описание инструмента
С помощью данного инструмента можно сгенерировать ASCII изображение с произвольной шириной ASCII символа по картинке. Каждый символ генерируемого изображения состоит из символов-точек шрифта Брайля и пробелов.

Параметры
Цветовой порог позволяет подобрать оптимальный уровень, при котором важные контуры оригинального изображения будут задействованы на генерируемой ASCII картинке.

Параметр ширина задает толщину отдельного символа, из которых состоит итоговая картинка.

Чтобы инвертировать основной цвет, на основе которого генерируются точки, нажмите на конпку "Инвертировать"

250

Ширина
